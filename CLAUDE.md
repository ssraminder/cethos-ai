# AgencyPro — CLAUDE.md

## Project
Professional digital marketing agency website with primary focus on political campaign marketing in India (Punjab elections), UAE, and Canada.
Positioning: "AI-Powered. Human-Managed. Results Guaranteed."

## Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS (custom tokens — see tailwind.config.ts)
- shadcn/ui (accessible primitives)
- Framer Motion 12 (animations)
- Lucide React (icons — ONLY icon library)
- Supabase (DB: scnmdbkpjlkitxdoeiaa / transalation-order project)

## Design System
See `design-system/MASTER.md` (generated by ui-ux-pro-max).
Colors: primary #EC4899 | secondary #F472B6 | cta #06B6D4 | bg-dark #0A0F1E | bg-light #FDF2F8
Fonts: Bebas Neue (display) | Plus Jakarta Sans (heading) | Source Sans 3 (body)

## Database
Supabase project: scnmdbkpjlkitxdoeiaa (transalation-order)
Table prefix: agp_ (14 tables — see supabase/migrations/001_agp_schema.sql)
Key table: agp_company_info — all brand identity (name, logo, favicon) is variable from this table.
Never hardcode agency name, logo, or favicon in code.

## Folder Structure
- app/ — Next.js App Router pages
- components/layout/ — Navbar, Footer, MobileMenu
- components/sections/ — Page-specific sections
- components/shared/ — Reusable UI components
- lib/context/ — CompanyInfoContext
- lib/data/ — Static seed data (fallback when DB is empty)
- lib/supabase/ — server.ts and client.ts
- lib/types.ts — All TypeScript interfaces
- design-system/ — MASTER.md + page overrides

## Coding Conventions
- PascalCase component filenames, named exports
- cn() from lib/utils.ts for conditional classNames
- Imports: external libs → @/lib → @/components → styles
- Icons: import individually from lucide-react. No emoji icons. No other icon libraries.
- All data fetching in Server Components. Client components only for interactivity.
- Contact form submission via Server Action

## Animation Rules
- Framer Motion whileInView with viewport={{ once: true }}
- Reveal duration: 0.5–0.7s ease-out
- Hover transitions: 300–400ms
- Stagger children: 0.1s delay per item
- Never animate without respecting prefers-reduced-motion (handled globally in globals.css)
- No decorative infinite animations except LogoStrip marquee

## Responsive Breakpoints
- Base: 375px (mobile-first)
- md: 768px (tablet)
- lg: 1024px (desktop)
- xl: 1440px (wide)

## Icon Rules
- Import from 'lucide-react' only
- No emojis used anywhere as icons
- Icon-only buttons must have aria-label
- Standard size: w-5 h-5 (inline), w-6 h-6 (standalone), w-8 h-8 (feature cards)

## Accessibility
- WCAG AA minimum (4.5:1 contrast)
- Focus states: focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-cta
- All images: descriptive alt text
- Keyboard navigable (logical Tab order)
- ARIA labels on icon-only buttons
